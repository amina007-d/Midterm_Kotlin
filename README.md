# Personal Task Tracker (Kotlin)

## Description
This is a console-based Personal Task Tracker application written in Kotlin.
The program allows users to manage tasks using a simple menu interface and
demonstrates object-oriented programming and functional concepts.

## Features
- Create, list, update, and delete tasks (CRUD)
- Task attributes: title, description, priority, status
- Filter tasks by status
- Sort tasks by priority
- Input validation and error handling
- In-memory data storage

## Technologies
- Kotlin
- Console-based application
- In-memory storage (MutableList)

## How to Run
1. Open the project in IntelliJ IDEA or any Kotlin-compatible IDE.
2. Ensure Kotlin/JVM is properly configured.
3. Run the `main()` function from `Main.kt`.
4. Interact with the application using the console menu.

## Notes
- All data is stored in memory and lost after program termination.
- Task IDs are generated automatically.
- Priority and status values are restricted using enums.
